What's all this then?

This tool generates a graphical score for   by   based on data collected in a spreadsheet from an analysis of the song's instrumentation and vocals.

What's with all the diamonds?

The song is made up of   verses, each with 16 beats and a 4⁄4 time signature. Each spoke of the diamond is one beat. Each of the diamond's concentric layers represents a different instrument, and at any point in the song where it is being played, the corresponding part of the diamond is illuminated. Each diamond represents one verse, and is read clockwise from the top.

What do the colours mean?

  Beat   Ooh
  Click   Hey
  Tone   Shout
  Steel   Laugh
  Beep
  Crunch

How do I work it?

Use the dropdown menu bottom right to view each of the song's verses.

Press the play button bottom left to see the diamonds change in time with the music. Press the stop button to pause the music and animation.

Press the subtitles button bottom left to toggle on and off the lyrics during the animation.

Press the clock button bottom left to toggle on and off the animated reveal in time with the beat.

Save each frame as an SVG using the download button bottom left.
